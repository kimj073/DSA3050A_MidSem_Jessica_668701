# DSA3050A_MidSem_Jessica_668701
A practical that focuses on cleaning and transforming a real-world dataset using Power Query and then building a professional Power BI dashboard to support business decision-making.

## Dataset

**Dataset Used:** [Download dataset](Dataset/all.xlsx)
This project uses the World Bank Projects & Operations dataset containing **5 worksheets**:
- World Bank Projects (28,018 rows)
- Themes (219,021 rows)
- Sectors (57,803 rows)
- GEO Locations (83,844 rows)
- Financers (31,979 rows)

## Business Problem
The World Bank needs to identify fraudulent loan disbursements, resolve critical data quality issues in its global project portfolio, and optimize geographic resource allocation across 28,000+ projects in 160 countries.
